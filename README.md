# Internship-project
💰 Expense Tracker Pro

A modern and responsive Expense Tracker web application built using HTML, CSS, JavaScript, Bootstrap, and Chart.js. It helps users manage their income and expenses with an interactive dashboard, charts, and local storage.

🚀 Features

- 📊 Dashboard with total balance, income, and expenses
- ➕ Add income and expense transactions
- 📅 Date and category selection
- 📈 Interactive charts using Chart.js
- 🌙 Dark/Light theme toggle
- 💾 Data stored using Local Storage
- 📱 Responsive design for mobile and desktop
- 🎨 Bootstrap-based UI components
- ✅ Form validation before adding transactions

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Chart.js
- Font Awesome

📂 Project Structure

Expense-Tracker-Pro/

▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click index.html or open it in any modern web browser.
4. Start adding your income and expenses.

📊 Project Functions

- Add new transactions
- Calculate total balance
- Display total income and expenses
- Show recent transactions
- View expense analytics using charts
- Switch between Dark and Light mode
- Store transaction data using Local Storage
- Clear all saved data
